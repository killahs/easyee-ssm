# Easyee SSM 企业级快速开发平台

## 介绍

这个项目主要 整合了广泛使用的 JaveEE 领域优秀框架（SSM）及 EasyUI 前端框架。它提供了基于用户、角色、权限方案的后台权限管理系统，安全管理框架及常用开发组件，为企业级项目开发提供了基础架构和规范。

## 使用技术

- Spring 4+(SpringMVC)
- MyBatis 3.4+
- Shiro 2+
- EasyMyBatis-Pagination
- EasyShiro
- EasyFilter
- EasyUI 1.4.3
- EasyUIEx 2.2.0

## 运行环境
- JDK 1.8
- MySQL 5.7
- Maven 3+
